# CriandoListaDeTarefas
<b>Criando lista de tarefas usado HTML, CSS e JavaScript.</b>

Projeto CRUD usando HTML, CSS e JavaScript.

Página de inclusão, alteração e exclusão de uma atividade via web.

Projeto simples, sem conexão (ainda) com Banco de Dados. Logo, não se pode recuperar as informações inclusas. Ficando disponível, apenas com a página aberta.

Usado para praticar scripts. Principalmente sobre Javascript.
